Steemit and DTube social extension

This module allow to display Steemit and DTube on the website. Links will be presented the same way as other social links in Odoo (icons). Icons will appear in website footer as well as on the blog.
Remember that you will get free updates with additional social media icons.

Add link to your social media

Simply fill in the fields in the Settings / Companies. No programming or technical knowledge is required.

Icons are displayed in the footer section

Same way as the original icons - when visitor clicks on the icon the social media website will be displayed.

Icons are displayed in the Blog

As on the photo - additional set of icons will be displayed in the blog section.
