Odoo - Coil Integration

This module adds Coil support for Odoo. It is also compatible with Multiwebsites.

What is coil?

Coil allows users to get paid for what they post online. "Whether you have 10 or 10 million followers, you can monetize your content using Coil". Coil is paying basing on the time spent on your website by Coil subscribers(subscription is 5$ a month).

Coil is supported by Ripple Labs and aims to change the model of Web Monetization.
Coil won't display any adds on your website and you can keep existing ads if you have some.

How to use module?

After signing up to the Coil.com you will get the content code (copy it from Settings/Creator Settings). You just need to paste the code in Odoo (Website/Configuration/Settings)
