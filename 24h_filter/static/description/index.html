24h filter

This module adds a new filter to 9 different views in Sales, Purchases, Inventory and Accounting.

Thanks to this module user will be able to filter variety of records basing on a date (last 24h). This might be useful especially when we have international transactions.

Today filter will be added to:
- Quotations (Sales/Quotations)
- Orders (Sales/Orders)
- Requests for Quotation (Purchases/Requests for Quotation)
- Purchase Orders (Purchases/Purchase Orders)
- Transfers (Inventory/Transfers)
- Customer Invoices (Invoicing/Customer Invoices)
- Payments (Invoicing/Sales/Payments and Invoicing/Purchases/Payments)
- Vendor Bills (Invoicing/Documents/Vendor Bills)
