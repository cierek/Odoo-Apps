Opportunity on Orders List

This module adds a feature of displaying opportunity on orders list view (Sales / Orders). Module was developed to save time of employees. They don't need to enter every single sales order to check if there is connected opportunity. Based on feedback it can save an hour per week per employee, which is around 50 hours of work a yesr, which can be used in more productive way.

Module is safe to use as it is just changing the list view.

How it works

Opportunity will be displayed next to the Sales Order it is connected to. In case there is no Opportunity for the Sales Order the blank space will be displayed.
