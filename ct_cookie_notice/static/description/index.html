Cookie Notice for Odoo

Cookie Notice is required in most of the EU Countries.
This simple module provides a cookie notice for Odoo.
It is not needed to enter any data or customise anything as everything comes in the package.

Cookie Notice will be displayed on your website

Popup information will be displayed on the bottom of the website. User will be informed about the cookies.
User can also see more information about cookies if needed
Cookie notice will be displayed also for backend users, in case they work on their our equipment they should also accept cookied.

If user clicks the "Learn More"

User will be directed to the safe and secure page, where he can see the video explaining what are the cookies.
User can also learn how to disable cookies if needed. Visit https://cookiesandyou.com/ for more details.
