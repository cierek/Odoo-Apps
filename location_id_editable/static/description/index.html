Source Location Zone changeable on delivery form

It sometimes happens that we want to:
- send a backorder from different Warehouse(for example if we have more than one warehouse)
- send part of the order from different warehouse
- send entire order from different warehouse(we realised the stock is in different warehouse)
To get this functionality it is important to have option of editing the "Source Location Zone" position.
Thanks to this module it is possible!

How it works?

Simply change the "Picking Type" in the "Additional Info" tab and Odoo will adjust the warehouse. Don't worry if you can't see the "Source Location Zone" - products will be shipped from the Warehouse assigned to the Picking Type.
