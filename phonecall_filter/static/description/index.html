Phone calls filters - today, last week

This module adds two new filters to the Scheduled Calls as well as to Logged Calls menu.

Thanks to this user will be able to filter calls basing on the creation date (Today, within Last Week)

How it works

New filters will be added to the filter list, on the tree view.

You need to install a free crm_phonecall module first.
